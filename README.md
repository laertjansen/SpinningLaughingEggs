An experimental project exploring an Arduino Uno, a continuous rotation servo, an LED RGB, plastic egg toys, and googly eyes.

What you'll need:

- Arduino Uno
- Continuous Rotation Servo
- RGB LED
- Plastic Egg Toys
- Googly Eyes
- Jumping wires
